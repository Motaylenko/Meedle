## 🚀 Як запустити

### Варіант 1: Автоматичний (Windows)
```
Подвійний клік на start.bat
```

### Варіант 2: Ручний
```powershell
# Terminal 1 - Backend
cd backend
npm install
npm run dev

# Terminal 2 - Frontend
cd frontend
npm install
npm run dev
```

### Варіант 3: Docker
```powershell
docker compose up --build
```

Після запуску відкрийте: **http://localhost:5173**
