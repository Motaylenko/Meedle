# Meedle

**Meedle** — це сучасна освітня платформа для студентів та школярів, яка створена як зручніша та функціональніша альтернатива системі Moodle. Проєкт спрямований на усунення недоліків застарілих інтерфейсів та впровадження інструментів для підвищення мотивації та організованості навчання.

## Основний функціонал

- **Персоналізація**: Повноцінна підтримка темної та світлої тем.
- **Система рейтингу**: Гейміфікація навчання, таблиці лідерів та відстеження прогресу.
- **Інтерактивний розклад**: Календар занять із системою фільтрації та push-сповіщеннями про зміни.
- **Кабінети**: Оптимізовані панелі керування для викладачів та студентів.

## Використані технології

Проєкт побудований як кастомний веб-застосунок з використанням мікросервісної архітектури.

- **Frontend**: React / Vue.js (SPA, адаптивний інтерфейс)
- **Backend**: Node.js (Express) / Python (FastAPI/Django)
- **База даних**: PostgreSQL
- **Web-сервер**: Nginx
- **Інфраструктура**: Docker, Docker Compose

## Структура проєкту

- `/frontend` — Код клієнтської частини
- `/backend` — Серверний код (API, бізнес-логіка)
- `/docker` — Конфігураційні файли для контейнерів
- `docker-compose.yml` — Файл для запуску всього стеку сервісів

## Як запустити проєкт локально

Для запуску проєкту необхідні встановлені **Docker** та **Docker Compose**.

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/Motaylenko/Meedle.git
   cd Meedle
   ```

2. **Запустіть застосунок:**
   Використовуйте Docker Compose для збірки та запуску всіх сервісів (web-app, api-server, postgres-db):
   ```bash
   docker-compose up --build
   ```

3. **Відкрийте у браузері:**
   Після успішного запуску перейдіть за адресою: `http://localhost:3000` (або інший порт, вказаний у конфігурації).
