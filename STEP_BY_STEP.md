## 🆘 Типові проблеми

### Проблема 1: "npm is not recognized"
```
❌ npm : The term 'npm' is not recognized...
```
**Рішення:** Встановіть Node.js з https://nodejs.org/

### Проблема 2: "EADDRINUSE"
```
❌ Error: listen EADDRINUSE: address already in use :::5000
```
**Рішення:** Порт зайнятий. Зупиніть інший процес або змініть порт.

### Проблема 3: Сайт не відкривається
```
❌ This site can't be reached
```
**Рішення:** 
1. Перевірте що обидва сервери запущені
2. Чекайте 10-15 секунд після запуску
3. Перевірте консолі на помилки

### Проблема 4: Білий екран
```
❌ Браузер показує білий екран
```
**Рішення:**
1. Відкрийте DevTools (F12)
2. Перевірте Console на помилки
3. Перевірте що backend запущений
